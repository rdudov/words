Телеграм-бот для изучения иностранных слов с акцентом на запоминание через постепенное усложнение проверки знаний. Система использует адаптивный алгоритм показа слов: сначала проверяет знание через выбор из вариантов, затем через самостоятельный ввод. Слова, которые пользователь знает хорошо (без ошибок), показываются реже и постепенно исключаются из активной практики.

Бот проверяет как перевод с родного языка на иностранный, так и наоборот.

Пользователь должен указать язык и свой уровень. Можно использовать несколько языков.

Список слов для изучения должен определяться автоматически, исходя из уровня пользователя, начиная с наиболее часто встречающихся.

Также пользователь может сам добавлять слова как на иностранном языке, так и на родном. 
При добавлении одного слова выводится его перевод, как в словаре (с примерами использования).

Учесть, что у слова может быть несколько значений, а также несколько форм (множественное, склонения, спряжения).

Пользователь может сам инициировать урок. Также бот должен пушить пользователя, если несколько часов не было уроков (в дневное время).

Нужно проработать пошагово все сценарии, в том числе реакцию на ошибки, опечатки и т.п.
